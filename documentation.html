First Deployment

Deploy your first Discord bot using our simple deployment process:

Prerequisites

  • A Discord bot token from the Discord Developer Portal
  • Your bot's source code (Node.js, Python, or other supported languages)
  • A VisiHost account with an active plan

Deployment Steps

Example Bot Structure 

// index.js
const { Client, GatewayIntentBits } = require('discord.js');

const client = new Client({
    intents: [GatewayIntentBits.Guilds, GatewayIntentBits.GuildMessages]
});

client.once('ready', () => {
    console.log(`Logged in as ${client.user.tag}!`);
});

client.on('messageCreate', message => {
    if (message.content === '!ping') {
        message.reply('Pong!');
    }
});

client.login(process.env.DISCORD_TOKEN);

Git Deployment

Deploy directly from your Git repository for seamless continuous deployment:

Git Integration 

# 1. Connect your GitHub/GitLab repository
git remote add visihost https://git.visihost.com/your-username/bot-name.git

# 2. Push your code
git push visihost main

# 3. Auto-deployment will trigger
# Your bot will be automatically deployed and started
Important: Make sure your repository includes a package.json (Node.js) or requirements.txt (Python) file for dependency management.

Environment Variables

Securely manage your bot's configuration using environment variables:

Setting Environment Variables 

# Via Dashboard
DISCORD_TOKEN=your_bot_token_here
DATABASE_URL=postgresql://user:pass@host:port/db
API_KEY=your_api_key

# Via CLI
visihost env:set DISCORD_TOKEN=your_token
visihost env:set DATABASE_URL=postgresql://...
